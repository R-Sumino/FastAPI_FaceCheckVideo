pyinstallerにて実行ファイルを生成したため正常に作動しない場合もございます。

実行方法
distフォルダ内のrun.exeファイルを起動
もしくは
コマンドプロンプトを開き、run.exeファイルを実行
localhost:8088を検索
接続できれば成功

・トップページ
中央のハンバーガーメニューを押すことで各メニューへ遷移できるボタンが出てくる
ログアウトボタンは未実装のため動作しない


・Facial Recognition、ストリーミング
カメラ映像から顔を検出および登録データと比較し、登録されているか確認する
カメラを選択し、更新ボタンを押すことで映像が出力される
強制停止ボタンによって映像を止めることができる
認証後の処理はまだ未実装のためこのページは現状ここまで


・Camera Information List、カメラ情報一覧
登録カメラの情報一覧
登録時のID、カメラ名称、カメラに接続するための情報、設置場所の情報が確認できる
各情報の右側にある鉛筆画像の青いボタンで情報を編集可能
ゴミ箱画像の赤いボタンで情報を削除可能
カテゴリを選択し、検索可能（検索ボタン未実装なためEnterキーで検索）
検索機能を利用後、Show Allボタンを押すことですべての情報が再度一覧できる


・Regist Camera、カメラの登録
カメラ情報を登録するページ
カメラの名前、設置場所、カメラの接続情報を入力し、登録ボタンを押すことでデータベースに登録可能
CameraTypeでWebカメラかIPカメラか選択後、それに応じた接続情報を入力する
登録後、登録完了ページが出る


・User Information List、ユーザ情報一覧
各ユーザの情報一覧
社員番号、名前、メールアドレス、登録画像枚数を確認できる
一覧右の緑のボタンを押すことで登録画像の詳細一覧が閲覧可能
一覧右の赤いボタンで情報削除可能
カテゴリを選択し、検索可能（検索ボタン未実装なためEnterキーで検索）
検索機能を利用後、Show Allボタンを押すことですべての情報が再度一覧できる


・ユーザデータ情報一覧
ユーザに紐づいた登録データ情報の一覧
画像パス、登録日時を確認できる
画像パスを押すことで登録した顔画像の確認ができる
上部にあるEditボタンから情報の編集可能
一覧の右の赤いボタンを押すことで情報削除可能
カテゴリを選択し、検索可能（検索ボタン未実装なためEnterキーで検索）
検索機能を利用後、Show Allボタンを押すことですべての情報が再度一覧できる


・Regist User、ユーザ登録
ユーザの登録を行うページ
ID（社員番号等）、名前、メールアドレスを入力し、Registボタンを押すことで登録可能
IDは4文字以上の半角英数字のみ
名前は４文字以上２０文字以下の半角英数字のみ
登録後、登録完了ページが出る


・Regist UserData、ユーザデータ登録
各ユーザに対応した顔画像の登録を行うページ
ユーザ選択し、画像をアップロードしてRegistボタンを押すと処理を行い登録をする
画像は複数枚アップロード可能
登録済みの顔画像は処理の段階で２重登録しないように弾いている
顔画像以外の画像は顔の切り抜きができないため登録されない
顔画像でも顔の検出ができなければ登録できない
登録後、登録完了ページが出る


トップページ以外のページでは右上にハンバーガーメニューを用意
ハンバーガーメニューから各ページに遷移可能
現段階ではURLに直接入力でも各ページに遷移可能


登録した顔画像は実ファイルとして保存している
保存先：files/ID_名前 の下にファイルが保存される
各ユーザのフォルダは顔画像データが登録されるときに自動で生成される
ユーザの情報全て削除する場合はフォルダ事削除
顔画像データのみを削除する場合は該当ファイルを削除


データベースにはカメラ用テーブル、ユーザ用テーブル、ユーザデータ用テーブルがある
ユーザデータ用テーブルには一覧に表示されてる情報以外に顔の特徴ベクトルとハッシュ値をバイナリデータとして持っている
